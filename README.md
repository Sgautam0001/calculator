## Requirements

You need:

- **Android Studio** (recommended) OR Android SDK Command Line Tools
- **Android SDK Platform Tools** (for `adb`)
- **Java (JDK)**  
  - This project works best with **Java 11** (because older Gradle/Android plugins may break with newer Java)
